# atom插件安装方法汇总

## 命令行安装

```
apm install pluginName
```

## 界面安装

> `Edit` -> `Preferences` -> `Settings` -> `Install`

## 手动安装

例如：

```
cd ~/.atom/packages
git clone https://github.com/emmetio/emmet-atom
cd emmet-atom
npm install --production --registry  http://registry.cnpmjs.org
```
